Created using Python3.4, and Biopython 1.64 for 64 bit.
The program name is REIDv0.1. (Restriction Enzyme Identify version 0.1)
Only fasta files.
The fasta file must be in the same directory as the program.
Enter the complete file name including the fasta extension.
Check the restriction enzyme you want, only one.
Click Go.
The program will print the location of the beginning of the RE sites.
If nothing is printed there are no sites.
